# LiveSplit Avalonia Master-Parity Ledger

This ledger tracks the file-by-file parity audit for `linux-port` against
`origin/master` and the vendored upstream component sources.

Accepted statuses:

- `Equivalent`: behavior is intentionally equivalent to master.
- `Accepted Divergence`: divergence is intentional and documented.
- `Bug`: current behavior diverges and must be fixed.
- `Missing`: master behavior has no Avalonia/core equivalent yet.
- `Obsolete Windows-only`: master behavior is not applicable to this port.

Accepted divergences are limited to the static dark Avalonia dialog theme,
modern Avalonia controls, Windows/Linux-only support, no macOS support, no Linux
auto-update UI, Wayland focus-only hotkeys, Linux memory injection/detour limits,
and platform file-association/window-manager limits.
